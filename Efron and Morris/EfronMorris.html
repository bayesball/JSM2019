The Model

The exchangeable model:

\[ y_j \sim Bin(n_j, p_j), j = 1, ..., 18 \] \[ \theta_j = logit(p_j) \sim N(\mu, \sigma), j = 1, ..., 18 \] \[ \mu \sim N(0, 2), \sigma \sim Cauchy(0, 1) \]

Fit the same model in JAGS

modelString <-"
model {
## sampling
for (i in 1:N){
y[i] ~ dbin(p[i], n[i])
logit(p[i]) <- theta[i]
theta[i] ~ dnorm(mu, tau)
}
## priors
mu ~ dnorm(0, 0.25)
tau <- 1 / (sigma * sigma)
sigma ~ dt(0, 1, 1) T(0, )
}
"
the_data <- list(y = d$r, n = d$n, N = 18)
posterior <- run.jags(modelString,
                      n.chains = 1,
                      data = the_data,
                      monitor = c("theta", "mu", "sigma"),
                      adapt = 1000,
                      burnin = 10000,
                      sample = 50000)
